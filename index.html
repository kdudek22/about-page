Projects:

  • AI powered survilance cameras for object detection

    Worked on a team of 3 people. We built an AI powered camera detection system using YOLO v8. We trained our model on wildlife data, with more than 15 classes. The model was run on a raspbery pi, connected to a camera. We set up a server, frontend app and a broker to handle events. The system recorded detections, and sent them to our server, and displayed it to a user. Later we hooked up a speker to the mqtt broker, and run dettering sound on detections.   • IOT air condition measurments

    Me and two friend joined together for a college project, the goal was to build a IOT solution. We got a CO2 sensor and a raspberry pi. We build an API in Java with Spring Boot, deployed it to an Azure VM, hosted a broker. We also build a mobile app for it in Kotlin with basic features like login, register, viewing current and historical data. The project succesfully passed us a subject.

  • P2P hash breaking

    For a college project we had to built a peer-to-peer hash breaking application, and include as many design patterns in it as we could. We did it in java using libp2p - a p2p networking framework. One of the nodes would broadcast the hash, that needs to broken, and the network would distribute tasks to all nodes. Once one node found the correct anwser it would distribute it across the system, and other nodes would verify it. It was my first expierience coding p2p programs, and it was fun but challenging
